Raised glass tile, for content that genuinely needs elevation (anagrafica head, tasse rate row, hero lozenge). Most list rows are NOT cards.

```jsx
<Card>…anagrafica…</Card>
<Card interactive>…clickable rate row…</Card>
<Card hero>…riepilogo media lozenge…</Card>
```

Props: `interactive` (hover lift), `hero` (brand-tinted focal tile), `padding`, `as`.
