Icon-only ghost button (40px square) for toolbar/header actions. Always provide an accessible `label`.

```jsx
<IconButton label="Cambia tema"><Moon size={18} /></IconButton>
<IconButton label="Esci"><LogOut size={18} /></IconButton>
<IconButton label="Copia" size="sm"><Copy size={16} /></IconButton>
```

Sizes: `sm` (32) · `md` (40) · `lg` (44). Hover fills with `--surface-2` and darkens the icon to `--ink`.
