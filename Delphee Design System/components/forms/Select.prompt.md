Labelled native select, matched to Input (well material, custom chevron).

```jsx
<Select label="Anno accademico" options={['2025/26', '2024/25', '2023/24']} />
<Select label="Esito" options={[{value:'all',label:'Tutti'},{value:'ok',label:'Superati'}]} />
```

Props: `label`, `options` (strings or `{value,label}`), plus native select attrs.
